## About Advent of Code

Advent of Code is an annual coding challenge that runs from December 1st to December 25th. Each day features a two-part programming puzzle that becomes progressively more challenging. It's a great way to improve problem-solving skills, learn new algorithms, and have fun with code during the holiday season.

## Approach

I aim to write clean, readable Python code that:
- Solves both parts of each day's puzzle
- Uses appropriate data structures and algorithms
- Includes comments explaining the logic
- Balances clarity with performance
